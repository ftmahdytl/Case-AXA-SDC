# AXA Insurance - Client Dashboard

Dashboard Flask multipage untuk presentasi Case 1: Top Performing Segment Analysis.

## Cara menjalankan

```bash
cd axa-dashboard-client
pip install -r requirements.txt
python app.py
```

Buka browser:

```text
http://127.0.0.1:5000/
```

## Data

Dashboard membaca file:

```text
data/AXA_Case1_Analysis_FINAL.xlsx
```

Pastikan file Excel memiliki sheet:

- Segment_COB
- Segment_Branch
- Segment_Channel
- COB_x_Branch
- COB_x_Channel
